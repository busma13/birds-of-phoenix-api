Birds of Phoenix API

Please use https://birds-of-phoenix-api.herokuapp.com/api/<bird-name> where <bird-name> is the name of the bird you are looking for. If you do not include a bird name, a list of the names of all the birds in the collection will be given.

Examples:

Request: GET https://birds-of-phoenix-api.herokuapp.com/api/red-tailed hawk

Response: 
            {
            "latin name": "Buteo jamaicensis",
            "family": "Raptors",
            "habitat": [
                "open woodlands",
                "thorn forest",
                "savanna"
            ],
            "size": {
                "lengthInches": 22,
                "widthInches": 53
            },
            "image": "https://upload.wikimedia.org/wikipedia/
                      commons/thumb/3/32/Red-tailed_Hawk.jpg/220px-Red-tailed_Hawk.jpg"
        }

Request: GET https://birds-of-phoenix-api.herokuapp.com/api/
